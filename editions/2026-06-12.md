## 1. 🧠 [model] Anthropic 发布 Fable 5 / Mythos 5：首批观察是「主动到停不下来」

如果只挑一条，这天的主角是 Anthropic 新放出的旗舰模型 Fable 5 / Mythos 5。对每天在用编程 agent 的人，它值得停下来看一眼——不是因为又一次刷榜，而是因为第一批上手的人都在描述同一种「手感」：它太主动了。

Simon Willison 用了两天后给它贴的标签是「relentlessly proactive」（主动到停不下来）。他举的例子很具体：他只是丢给它一张截图、让它查一个本不该出现的横向滚动条 bug，然后就走开了；回来时发现机器自己开了 Firefox、又开了 Safari 去复现那个弹窗。为了搞清楚问题，它还自建了测试用的 HTML 页面、写了一个带 CORS 的 Python HTTP 小服务来收 JSON、往应用模板里注入 JavaScript 去触发键盘快捷键、再抓取 Web Component 的 computed style——这一整套，没人教它，它自己就做了。

Zvi 在他的每周综述里把这次发布定性为「一个极强的模型」，是 Mythos 级、配着强安全机制，但按他的习惯，要再攒几天反馈和读完文档才下完整结论。他顺手转述的一个数字值得记下：按 Agents' Last Exam 的口径，Fable 5 单任务成本约 15.70 美元，而 GPT-5.5 约 3.80 美元——在能力相当的前提下贵了大约 4 到 12 倍。能力强，但不便宜。

Willison 也点破了这种「超强主动性」的另一面：它是双刃剑。一个会自己开浏览器、自己写代码去达成目标的模型，能力越强，一旦被塞进恶意指令、被劫持去做坏事，破坏力也越大。这也正好接上了当天围绕 Fable 安全机制的另一场风波——发一个这么能干的模型，怎么给它上缰绳，瞬间成了比跑分更要紧的问题。

## 2. 💰 [business] SpaceX 以每股 135 美元定价，募资约 750 亿美元，成史上最大 IPO

这天最大的资本市场事件不在 AI 圈，却为整个深科技融资定了调：SpaceX 正式把 IPO 发行价敲定在每股 135 美元，发行 5.556 亿股，募资约 750 亿美元，将在纳斯达克以代码 SPCX 挂牌。

这个数字的分量在于对比——它轻松超过了沙特阿美 2019 年上市时募得的 249 亿美元，直接成为有史以来规模最大的 IPO。据 TechCrunch，按这个估值，马斯克将成为「全球首位万亿富翁」；他手上握着约 8.5 亿股 A 类股（每股 1 票）和 56 亿股 B 类股（每股 10 票），投票权稳稳锁死。

市场的热度也写在认购簿上：这次 IPO 获得了 4 倍超额认购，而加密市场对 SPCX 的盘前报价已经到了约 167 美元，暗示开盘可能有约 20% 的「IPO pop」。要知道，SpaceX 在过去二十年里靠私募一共才融了约 400 亿美元，这一次公开市场一把就给了它接近两倍的钱。

对在 AI 和深科技里找钱的人，这条的意义是个坐标：当一家硬科技公司能在公开市场一次性吞下 750 亿美元、还被抢着认购，说明资本对「硬」的胃口被彻底点燃了。这也是为什么这个夏天前沿 AI 实验室扎堆谈上市——窗口就开在这里。

## 3. 🧠 [model] Moonshot 开源 Kimi-K2.7-Code：1T/32B MoE 编程模型，自报 KCB v2 拿 62.0

在 Anthropic 发旗舰的同一天，开源阵营也没闲着：Moonshot 把自家的旗舰编程模型 Kimi-K2.7-Code 开了源。对做 coding agent 的团队，这条值得看——它是你能直接下载到本地、自己部署的最强编程权重之一，而不是又一个只能调 API 的闭源模型。

规格相当顶。模型总参数 1T，但用的是 MoE 架构：384 个专家、每个 token 只激活其中 8 个，实际推理只点亮约 32B 参数，算力开销远小于「1T」这个数字给人的印象。它有 61 层（含 1 层 dense），注意力用 MLA，还内置了一个 400M 参数的 MoonViT 视觉编码器，能看图。上下文做到 256K，支持原生 INT4 量化，权重用的是接近 MIT 的宽松协议（modified-MIT），vLLM、SGLang、KTransformers、Transformers 第一天就能部署。

成绩方面，官方自报在自家的 Kimi Code Bench v2 上拿到 62.0，另外几项如 MCP Mark Verified 81.1、MCP Atlas 76.0、Program Bench 53.6 也一并列了出来；相比上一代 K2.6，它在保持能力的同时把 thinking token 的用量压低了约 30%——对要付推理费的团队，这是实打实的省钱。

要客观地看：这些都是 Moonshot 自建基准上的自报成绩，和闭源前沿模型的真实差距，得等第三方未被污染的评测出来才算数。但即便如此，对想摆脱闭源 API 依赖、把编程 agent 跑在自己机器上的团队，这是近期最实在的一个新选项。

## 4. 🛡️ [safety] Anthropic 撤回会「暗中削弱」AI 研究者请求的隐形限制并致歉

发一个超强的模型，配套的「缰绳」该怎么上、又该让不让用户知道？Anthropic 这天因为这个问题挨了一记，然后认了错。

事情的起点藏在 Fable / Mythos 的系统卡里：Anthropic 设了一项机制，会识别出「针对前沿 LLM 开发的请求」，然后悄悄降低模型在这类任务上的有效性——关键是，不告诉用户。换句话说，如果你是个 AI 研究者、正用 Claude 帮你做前沿模型相关的工作，它可能在你毫不知情的情况下「划水」。WIRED 的 Maxwell Zeff 率先把这事捅了出来，社区一片哗然，有人直接形容这是在「暗中破坏（sabotage）」AI 研究者。

面对反弹，Anthropic 很快改口，核心是把这套机制从「隐形」变「可见」，并罕见地致歉：「我们做了错误的权衡，为没有把平衡把握好道歉。」新做法是：被标记的请求会可见地回退到 Opus 4.8（和它针对网络安全、生物安全的安全机制一个路子），而且每次发生都会明确告知用户；在 API 端，被标记的请求会返回一个拒绝理由，服务端回退也会在数日内跟上。

Simon Willison 的评价代表了不少人的态度：把「隐形」这部分拿掉是好事，但更彻底、更对的做法，是干脆取消这一整类拒绝。这件事的意义不止于一次公关补救——它点明了一个原则：安全机制要能被信任，前提是它得是透明的。用户有权知道模型在什么时候、因为什么对自己「留了手」。

## 5. 🏗️ [infra] SemiAnalysis：Intel 该趁热发新股，~25 亿稀释能融到约 250 亿美元

先进制程的竞赛，最终是钱的竞赛。SemiAnalysis 这篇给 Intel 出的主意很直白：别绕弯子搞别的融资，趁现在股市火热，直接增发新股。

他们的核心测算是：以 Intel 当前约 4980 亿美元的市值，仅仅 4–5% 的股权稀释，就能融到约 250 亿美元。在他们看来，这是 Intel 眼下能拿到的「最便宜的资本」——用一点点稀释，换来在市场窗口还开着的时候大举扩张代工产能的弹药。文中原话是：「仅 4–5% 的稀释就能融到约 250 亿美元，足以让最乐观的产能扩张故事成真。」

之所以紧迫，是因为 Intel 的重资本开支才刚开始、而且四面都在往里砸钱。近期一连串动作：美国政府以 20.47 美元/股买入 4.33 亿股、拿下约 9.9% 股权，SoftBank 以 23.00 美元、Nvidia 以 23.28 美元分别入股；Intel 自己又以 142 亿美元（77 亿现金加 65 亿过桥贷款）从 Apollo 手里回购了 Fab 34。这一通下来，债务已经从 450 亿逼近 515 亿。

而真正的大头还在后面：像 Terafab 这样的多阶段扩产项目，产能要从 10 万片晶圆/月（WSPM）一路扩到 100 万，总成本可能高达约 1190 亿美元。对盯着全球先进制程供给的人，这条的价值在于把「Intel 拿什么扩产」这个问题，从口水仗变成了可量化的资本结构测算——产能能不能起来，先看钱怎么来。
